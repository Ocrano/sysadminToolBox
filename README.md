# ToolboxPyQt6

Toolbox de gestion infrastructure avec PyQt6

## Installation

```bash
pip install -r requirements.txt
```

## Utilisation

```bash
python main.py
```

## Structure du projet

```
src/
├── core/           # Composants centraux
├── ui/             # Interface utilisateur
│   ├── dialogs/    # Dialogues
│   └── tabs/       # Onglets
├── handlers/       # Gestionnaires de services
├── network/        # Fonctionnalités réseau
└── utils/          # Utilitaires
```

## Build

```bash
cd build
python build_exe.py
```
